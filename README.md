# NodeJS-MongoDB-API

A RESTful API built with Node.js, Express.js, MongoDB Atlas, Mongoose, and JWT Authentication.

## Features

- User Signup
- User Signin
- JWT Authentication
- MongoDB Atlas Integration
- REST API

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Postman

## Installation

```bash
git clone https://github.com/sanaperween602/NodeJS-MongoDB-API.git
cd NodeJS-MongoDB-API
npm install
node index.js
```

## API Endpoints

### Signup

```
POST /signup
```

### Signin

```
POST /signin
```

### Create Todo

```
POST /todo
```

### Get Todos

```
GET /todos
```
